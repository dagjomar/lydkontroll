# Support

Lydkontroll is an early source-available project supplied without warranty,
guaranteed response times, or free individual support.

- Use GitHub issues for reproducible bugs and narrowly scoped feature requests.
- Use the private process in `SECURITY.md` for vulnerabilities.
- Include the commit, macOS/iOS versions, expected behavior, actual behavior,
  logs with secrets removed, and minimal reproduction steps.

The supported target is Apple Silicon macOS with iPhone Safari for optional
mobile control. Android, Windows, Intel Macs, ordinary-LAN control, cloud relay,
and unsigned downloadable public bundles are not currently promised.

Commercial licensing, finished app bundles, setup, or paid support may be
offered separately by the project owner in the future. Their availability is
not guaranteed by this repository.
